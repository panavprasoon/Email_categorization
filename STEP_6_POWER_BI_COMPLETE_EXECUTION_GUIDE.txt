- Columns shelf = horizontal axis.

SECTION 5: CALCULATED FIELDS + FORMATTING
================================================================================

RULE BEFORE YOU START
---------------------
- In Tableau, calculations are created per data source.
- Make sure you are using the correct data source before creating a field.
- When you create a calculated field, Tableau saves it in the selected data source.

5.1 Create calculated fields in CoreModel
-----------------------------------------
Open any worksheet using CoreModel first.

How to execute:
- Click a worksheet tab, or create a new worksheet
- In the Data pane, confirm the selected data source is CoreModel
- Right-click empty area in the Data pane
- Click Create Calculated Field

A) Prediction Date

Field name:
Prediction Date

Formula:
DATE([Prediction Timestamp])

B) Month Start

Field name:
Month Start

Formula:
DATETRUNC('month', [Prediction Date])

C) Month-Year Label

Field name:
Month-Year Label

Formula:
LEFT(DATENAME('month', [Month Start]), 3) + ' ' + STR(DATEPART('year', [Month Start]))

D) Confidence Category

Field name:
Confidence Category

Formula:
IF [confidence] >= 0.8 THEN
    'High (>=80%)'
ELSEIF [confidence] >= 0.5 THEN
    'Medium (50-79%)'
ELSE
    'Low (<50%)'
END

E) Processing Speed Bucket

Field name:
Processing Speed Bucket

Formula:
IF [processing_time_ms] < 100 THEN
    'Fast (<100ms)'
ELSEIF [processing_time_ms] < 300 THEN
    'Normal (100-299ms)'
ELSE
    'Slow (>=300ms)'
END

F) Feedback Date

Field name:
Feedback Date

Formula:
DATE([Feedback Timestamp])

5.2 Core KPI calculations in CoreModel
--------------------------------------
A) Total Predictions

Field name:
Total Predictions

Formula:
COUNT([Prediction ID])

B) Avg Confidence

Field name:
Avg Confidence

Formula:
AVG([confidence])

C) High Confidence Count

Field name:
High Confidence Count

Formula:
SUM(IIF([confidence] >= 0.8, 1, 0))

D) Low Confidence Count

Field name:
Low Confidence Count

Formula:
SUM(IIF([confidence] < 0.5, 1, 0))

E) Total Corrections

Field name:
Total Corrections

Formula:
COUNT([Feedback ID])

F) Correction Rate

Field name:
Correction Rate

Formula:
IF [Total Predictions] = 0 THEN
    0
ELSE
    [Total Corrections] / [Total Predictions]
END

G) Success Rate

Field name:
Success Rate

Formula:
1 - [Correction Rate]

H) Active Model Accuracy

Field name:
Active Model Accuracy

Formula:
AVG(IIF([is_active] = TRUE, [accuracy], NULL))

If Tableau says TRUE is invalid because is_active imported as text, use this instead:

AVG(IIF(LOWER(STR([is_active])) = 'true', [accuracy], NULL))

I) Active Model F1

Field name:
Active Model F1

Formula:
AVG(IIF([is_active] = TRUE, [f1_score], NULL))

If Tableau says TRUE is invalid because is_active imported as text, use this instead:

AVG(IIF(LOWER(STR([is_active])) = 'true', [f1_score], NULL))

J) Avg Processing Time (ms)

Field name:
Avg Processing Time (ms)

Formula:
AVG([processing_time_ms])

5.3 Operational calculations in Operations data source
------------------------------------------------------
Switch to a worksheet that uses Operations.

A) API Total Requests

Field name:
API Total Requests

Formula:
COUNT([Audit Log ID])

B) API Success Rate

Field name:
API Success Rate

Formula:
IF [API Total Requests] = 0 THEN
    0
ELSE
    SUM(IIF([status_code] >= 200 AND [status_code] < 300, 1, 0)) / [API Total Requests]
END

C) API Avg Latency (ms)

Field name:
API Avg Latency (ms)

Formula:
AVG([latency_ms])

D) API Errors

Field name:
API Errors

Formula:
SUM(IIF([status_code] >= 400, 1, 0))

E) Audit Date

Field name:
Audit Date

Formula:
DATE([Audit Timestamp])

SECTION 16: TROUBLESHOOTING
================================================================================

Problem: Tableau Public sign-in fails
-------------------------------------
Fix steps:
1) Verify you completed email verification
2) Reset password if needed
3) Try signing in at https://public.tableau.com/ first
4) Then return to the desktop app

Problem: CSV export command fails
---------------------------------
Fix steps:
1) Confirm the virtual environment is active
2) Confirm pandas is installed
3) Confirm DB connection works with the DB OK command
4) Re-run the export command carefully without editing quotes

Problem: Charts show blank values
---------------------------------
Fix steps:
1) Confirm the underlying CSV has rows
2) Confirm the worksheet is using the correct data source
3) Confirm relationships in CoreModel are correct
4) Remove filters temporarily to see if they are hiding all rows
5) Check calculated field names and spelling

Problem: Active model calculations return blank
-----------------------------------------------
Fix steps:
1) Verify model_versions has one active row
2) Check whether is_active imported as boolean or text
3) Use the alternate TRUE or text formula from Section 5 if needed

Problem: Month labels sort incorrectly
--------------------------------------
Fix steps:
1) Use Month Start on the axis instead of Month-Year Label
2) Use the label only for display if necessary

Problem: Published viz does not refresh when database changes
-------------------------------------------------------------
Fix steps:
1) Re-run the CSV export command
2) Open the workbook
3) Refresh or reconnect if needed
4) Publish again to Tableau Public

Problem: Published viz should not be public
-------------------------------------------
Reality:
- Tableau Public is not the right place for private data
- Keep the workbook local or use a paid or private platform instead
